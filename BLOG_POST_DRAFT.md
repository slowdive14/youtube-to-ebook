# [블로그 초안] 영미권 유튜버의 영상을 '한글 전자책'으로 자동으로 만들어 읽는 법 (코딩 몰라도 됨)

안녕하세요! 오늘은 제가 직접 수정해서 쓰고 있는 아주 편리한 도구를 하나 소개하려고 합니다.

## 🤔 왜 만들게 되었나? (개발 배경)

유튜브에는 정말 좋은 정보가 많습니다. 특히 앤드류 후버만(Andrew Huberman)이나 영미권의 깊이 있는 인터뷰 영상들은 내용이 정말 알차죠. 하지만 이런 영상들은 몇 가지 문제가 있습니다.

1.  **시간이 너무 오래 걸립니다:** 2시간짜리 팟캐스트를 다 듣기엔 우리 시간이 부족합니다.
2.  **영어의 압박:** 자막이 있어도 계속 화면을 봐야 하고, 뉘앙스를 온전히 이해하기 피곤할 때가 있습니다.
3.  **다시 찾아보기 힘듭니다:** "그때 그 영상에서 비타민 D 얘기가 뭐였지?" 하고 다시 찾으려면 영상을 뒤져야 합니다.

**"그냥 누가 이 영상 내용을 깔끔한 **한글 매거진 기사**처럼 요약해서, 내 **핸드폰 이북 리더**에 쏙 넣어주면 안 될까?"**

이런 생각으로 찾아보니 이미 [zarazhangrui님의 훌륭한 오픈소스 프로젝트](https://github.com/zarazhangrui/youtube-to-ebook)가 있더군요. 

하지만 원본은 영어 전용이고 유료 API인 Claude를 사용해야 해서, **제가 한국인에게 맞게, 그리고 무료로 쓸 수 있는 구글 제미나이(Gemini)를 사용하도록 코드를 수정했습니다.**

개발자가 아니어도 누구나 10분이면 나만의 **'영상 도서관'**을 만들 수 있습니다. 차근차근 따라와 보세요!

## 🛠️ 준비물

딱 두 가지만 컴퓨터에 설치되어 있으면 됩니다.
1.  **Git (깃):** 프로그램을 내 컴퓨터로 가져오는 배달부입니다. ([다운로드](https://git-scm.com/downloads)) -> 설치할 때 그냥 계속 `Next`만 누르면 됩니다.
2.  **Python (파이썬):** 프로그램을 실행하는 엔진입니다. ([다운로드](https://www.python.org/downloads/)) -> **주의:** 설치 첫 화면에서 **`Add Python to PATH`** 체크박스를 꼭 켜주세요!

## 1단계: 내 보물창고 만들기 (Fork)

남이 만든 코드를 내 것으로 가져오는 단계입니다.

1.  [이 깃허브 페이지](https://github.com/slowdive14/youtube-to-ebook)로 이동합니다.
2.  오른쪽 위에 있는 **`Fork`** 버튼을 누릅니다.
3.  `Create fork`를 누르면 내 계정으로 이 프로그램이 복사되어 옵니다.

## 2단계: 내 컴퓨터로 가져오기

이제 윈도우의 검은 화면(터미널)을 열 차례입니다. 겁먹지 마세요!

1.  `윈도우 키` + `R`을 누른 뒤 `cmd`라고 치고 엔터를 누릅니다.
2.  우선 프로그램을 저장할 폴더(예: 문서 폴더)로 이동한 뒤, 아래 명령어를 순서대로 입력합니다.
    ```bash
    cd Documents
    git clone https://github.com/slowdive14/youtube-to-ebook.git
    cd youtube-to-ebook
    ```
3.  필요한 부품들을 설치합니다.
    ```bash
    pip install -r requirements.txt
    ```
    (뭔가 주르륵 설치되면 성공입니다!)

## 3단계: 무료 AI 열쇠 받기 (API Key)

이 프로그램은 구글의 최신 AI '제미나이'를 사용합니다. 무료로 키를 발급받을 수 있습니다.

1.  **[Google AI Studio](https://aistudio.google.com/app/apikey)**에 접속해서 로그인합니다.
2.  `Create API key`를 눌러서 키를 발급받고 복사해 둡니다. (메모장에 붙여넣어 두세요)
3.  폴더 안에 있는 `.env.example` 파일의 이름을 `.env`로 바꿉니다. (뒤에 `.example` 지우기)
4.  메모장으로 `.env` 파일을 열고 `GEMINI_API_KEY=` 옆에 아까 복사한 키를 붙여넣습니다.

*(선택사항)* 유튜브 API 키도 필요한데, 이건 [구글 클라우드 콘솔](https://console.cloud.google.com/)에서 받을 수 있습니다. 조금 복잡하면 구글에 "유튜브 API 키 발급"이라고 검색해보세요!

## 4단계: 좋아하는 유튜버 등록하기

어떤 채널을 구독할지 정해야겠죠?

1.  폴더에 있는 `channels.txt` 파일을 메모장으로 엽니다.
2.  평소 즐겨보는 유튜버의 핸들(골뱅이 아이디)을 적습니다.
    ```text
    @hubermanlab
    @DrChatterjeeRangan
    ```

## 5단계: 버튼 누르기 (실행!)

모든 준비가 끝났습니다. 다시 검은 화면(cmd)으로 돌아가서 마법의 주문을 외웁니다.

```bash
python main.py
```

이제 프로그램이 알아서 최신 영상을 찾아내고, 자막을 읽고, 한글 기사로 바꾸고, 전자책(EPUB)으로 묶어줍니다.

완성된 파일은 `newsletters` 폴더 안에 예쁜 전자책 파일로 저장됩니다. 이걸 스마트폰으로 보내서 '리디북스'나 '애플 도서' 같은 앱으로 열어보세요. 지하철 출근길이 훨씬 유익해질 거예요!
